Combobox

BONIC Widgets
Este widget funciona como un combobox en donde se selecciona un elemento y a la hora de seleccionar se muestra una descripción del mismo
Para crear opciones se necesita agregar hijos al tag con la clase 
.combobox-options
 con la siguiente estructura: 
<div category="2" > Nombre de la opción
    <div class="combobox-description" contenteditable="true">
        Descripción de la opción
    </div>
</div>
Cuando existe una opción seleccionada se le agrega selected como atributo

El atributo de la categoría define un color al cual se asocia la opción, los colores de cada categoría se definen en el css
