Meter

BONIC Widgets
El meter tiene como función medir entre un predeterminado rango de valores y en este caso informar si sobre pasa un limite establecido
Para la configuración existe una serie de variables de css
  1. --percent: define la posición de la aguja
  2. --maxrange: define el rango máximo
  3. --minrange: define el rango mínimo
  4. --max-alert: define el valor máximo antes de mostrar la alerta
