Toggle

BIONIC Widgets
El toggle funciona para activar o desactivar alguna opción en un formulario
Para el elemento existe un atributo llamado side, que determina si el elemento está a la izquierda o a la derecha 
<div class="toggle" side="l">
    <div class="circle"></div>
</div>
<div class="toggle" side="r">
    <div class="circle"></div>
</div>
<div class="toggle" side="l"
    onclick="(() => {this.setAttribute('side', this.attributes.side.value == 'l'? 'r': 'l')})()">
    <div class="circle"></div>
</div>
Si le agregamos funcionalidad con js, se puede realizar que el elemento cambie el atributo con un click
